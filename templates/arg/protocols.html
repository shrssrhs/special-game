// Протоколы взаимодействия

Внутренние инструкции для сотрудников департамента. Публикуются в ограниченном объёме.

ПРОТОКОЛ 1-А
Первичная идентификация объекта

1. При обнаружении объекта — немедленно зафиксировать время, координаты, количество свидетелей.

2. Не сообщать объекту о факте наблюдения.

3. Не называть объекту своего настоящего имени.

4. Если объект назвал ваше имя первым — немедленно покинуть локацию.
Не уточнять, откуда он его знает.

5. Составить донесение не позднее чем через 6 часов после контакта.
Если донесение уже составлено до вашего возвращения — обратиться в отдел Б.

ПРОТОКОЛ 2-В
Действия при дублировании записей

1. При обнаружении двух идентичных записей об одном событии — сохранить обе.

2. Не уничтожать ни одну из копий до завершения верификации.

3. Верификацию проводит исключительно сотрудник, не участвовавший в составлении ни одной из записей.

4. Если верифицирующий сотрудник сообщает, что обе записи составил он — ██████████████████

ПРОТОКОЛ 4-Б
Действия при обнаружении объекта класса «Подражатель»

1. Не вступать в визуальный контакт более 3 секунд.

2. Не оборачиваться повторно после первого контакта.

3. При попытке объекта имитировать знакомое лицо —
задать контрольный вопрос (см. Приложение В).

4. Не задавать контрольный вопрос дважды.
Объект учится.

5. После контакта немедленно сверить с коллегой наличие у себя всех личных документов.
Коллегой должен быть человек, которого вы знаете не менее пяти лет.

Приложение В изъято из публичного доступа.

ПРОТОКОЛ 7
Завершение наблюдения

1. Считать наблюдение завершённым только при наличии подписи двух независимых наблюдателей.

2. Если наблюдатель не может подтвердить завершение — наблюдение не считается завершённым.

3. Если наблюдатель не может быть найден — ████████████████

4. Если наблюдатель был найден, но не является собой — см. Протокол 4-Б.
